Privacy Policy

Last updated: July 2026

Archive is a local-first browsing memory tool. Your data stays on your machine unless you choose to run an optional self-hosted backend.

What we collect

Archive saves text you explicitly view on supported websites, including:

  • Post or result text
  • Author name (when visible on the page)
  • Permalink URL (when available)
  • Platform name and capture timestamp

Where data is stored

  • Browser extension (default): all memories stored in IndexedDB on your device. No server required.
  • Optional local backend: if you run the Python server, saves are mirrored to your local PostgreSQL database.
  • No cloud storage is used by default.

What we do not collect

  • Passwords or credentials
  • Private messages
  • Payment information
  • Browsing history outside supported platforms
  • Data from sites not listed in the extension permissions

Update emails

If you subscribe on our website, we store your name and email only to send product updates. You can unsubscribe anytime. We do not use that list for ads.

Third parties

Archive does not sell or share your browsing memories. The extension communicates only with:

  1. Supported content pages (read-only DOM extraction on sites you visit)
  2. Your browser's local IndexedDB storage (default — no network)
  3. Your optional local Archive backend (http://127.0.0.1:8000) — only if you run it
  4. OpenAI (optional, only if you enable it in a self-hosted backend)

Data control

  • Extension-only (default): uninstall Archive, or clear extension/site data in Chrome settings.
  • Optional backend: delete the PostgreSQL archive database.
  • Uninstall the extension to stop new captures.
